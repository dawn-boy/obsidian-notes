# Network connectedness
- A state in network where a path exists between two nodes and that there are no isolated individuals or groups.
- This is essential for complete transfer of information across a network

# Bifurcation
- The division of a network into two or more independent sub-groups. Nodes in one component cannot reach another.
- This represents a failure in network side communication.
- But, this is highly improbable in randomly formed networks

# Uniform-ness at random selection
- The process where each potential choice has an equal probability of being selected from the available options. 
- This method of creating connections ensures an unbiased model 
- It also reveals the natural tendency for networks to become interconnected

# Improbable event
- An outcome that is theoretically possible but has an extremely low probability in practice
